HTML5, WebSockets & server push

"The Web, asynchronous"

Er is veel gebeurd sinds Tim Berners-Lee in 1989 HTTP en HTML voorstelde. "Het web" begon statisch, daarna gegenereerd, toen dynamisch, eventjes "2.0" en inmiddels asynchroon. Javascript is een "first class citizen" geworden, en HTML - altijd al een levende specificatie - is in versie 5 een waaier geworden met voor ieder wat wils. Deze cursus gaat in op diverse onderdelen ervan die onder de noemer "asynchroon" vallen. Er wordt stilgestaan bij de mate van ondersteuning door browsers & servers. Alle deze nieuwe technologieën worden tastbaar gemaakt met een demo, codevoorbeelden en hands-on lab, waarin we zowel met client als server code aan de gang gaan. Je zult zien dat je er al snel zelf mee aan de slag kunt!

Wil je ook alles weten van WebSockets, Servlet 3.x, WebRTC, SSE, Comet, Atmosphere, SPDY en HTTP2? Geef je snel op!
