Beste cursisten,

Over een week is het al zover! De eerste avond van de 2e cursussessie van de cursus HTML5, Websockets & server push gaat van start. Hierbij ontvang je alvast een sneak preview van de sheets en het cookbook.

De cursus staat gepland voor 3 (drie) avonden. De cursisten - jullie dus - staan aan het roer! Het is straks aan jullie om, Scrum style, middels prioritering aan te geven welke onderwerpen in welke volgorde behandeld gaan worden, hoe diepgaand en op welke manier.

Dit jaar voor het eerst staat al het materiaal op een private github repo van het Ordina account. Laat me aub je github id weten, dan maak ik je reader. Je haalt al het materiaal binnen met:
 git clone https://github.com/OrdinaNederland/cursus-websockets-server-push.git

Installeer alvast de volgende software op je laptop:
JDK8
Git
Maven
IDE naar keuze - Eclipse, IntelliJ of NetBeans; als je er maar zelf goed mee overweg kan
NodeJs 5+
Glassfish 4.1
Wireshark
Curl
Fiddler
Firefox en/of Chrome

Voor details, download URLs etc: zie het cookbook (git repo: docs\WebSockets etc. hands-on lab cookbook.docx). Loop je vast, laat me het dan weten!
NB Ikzelf werk op Windows 7 (eigen image; geen Ordina laptop) met Firefox, cygwin en IntelliJ. De workshop software werkt ook op andere combinaties. Bij de eerste cursussessie is 'YMMV' echter wel bewaarheid gebleken. Met name Linux en NetBeans gaven wat probleempjes. Neem het gerust mee; neem dan wel wat zelfredzaamheid mee!

Mocht je onverhoopt niet kunnen, laat het dan aub per ommegaande weten aan Monique of aan mij, dan kunnen we iemand van de wachtlijst jouw plek aanbieden!

De eerste avond is op dinsdag 5 april, 16:00-21:00u, A12.5, Ordina Nieuwegein.

Je bent om 15:00u al welkom bij de koffieautomaat op A12 voor een 'install fest' mocht je daar nog niet eerder aan toegekomen zijn.
